## 2. proto 包详细设计

**文件：** `proto/proto.go`

### 2.1 常量定义

```go
package proto

const (
    TypeHeartbeat    uint8 = 0x01  // client → server 心跳
    TypeHeartbeatAck uint8 = 0x02  // server → client 心跳回包
    TypeDataReq      uint8 = 0x03  // server → client 业务请求数据
    TypeDataRsp      uint8 = 0x04  // client → server 业务响应数据
    TypeClose        uint8 = 0x05  // 双向 关闭通知
)

// 帧头固定长度：4(len) + 4(reqID) + 1(type) = 9 字节
const HeaderSize = 9

// 单帧 payload 最大 4MB，防止内存攻击
const MaxPayloadSize = 4 * 1024 * 1024
```

### 2.2 Frame 结构体

```go
type Frame struct {
    ReqID   uint32
    Type    uint8
    Payload []byte  // 长度 0 ~ MaxPayloadSize
}
```

### 2.3 WriteFrame 函数

**功能：** 将 Frame 序列化并写入 w，调用方负责并发安全（加写锁）。

**算法：**
```
buf[0:4]  = BigEndian(len(Payload))
buf[4:8]  = BigEndian(ReqID)
buf[8]    = Type
buf[9:]   = Payload
w.Write(buf)
```

**错误：** 底层 IO 错误直接返回，不做重试。

### 2.4 ReadFrame 函数

**功能：** 从 r 读取一帧，返回 Frame。

**算法：**
```
io.ReadFull(r, header[9])
payloadLen = BigEndian(header[0:4])
if payloadLen > MaxPayloadSize → return error
reqID      = BigEndian(header[4:8])
frameType  = header[8]
payload    = make([]byte, payloadLen)
io.ReadFull(r, payload)
return Frame{reqID, frameType, payload}
```

## 3. tlsutil 包详细设计

**文件：** `tlsutil/tlsutil.go`

### 3.1 GenerateSelfSigned

```go
func GenerateSelfSigned() (certPEM, keyPEM []byte, err error)
```

**算法：**
1. `rsa.GenerateKey(rand.Reader, 2048)` 生成私钥
2. 构造 `x509.Certificate`：
   - `SerialNumber`: 随机大整数
   - `Subject.CommonName`: `"ts-server"`
   - `NotBefore`: `time.Now()`
   - `NotAfter`: `time.Now().Add(10 * 365 * 24 * time.Hour)`（10年）
   - `KeyUsage`: `KeyUsageCertSign | KeyUsageDigitalSignature`
   - `IPAddresses`: `[0.0.0.0]`（支持任意 IP）
   - `DNSNames`: `["ts-server", "localhost"]`
   - `IsCA`: true，`BasicConstraintsValid`: true
3. `x509.CreateCertificate` 自签
4. PEM 编码 cert 和 key 返回

### 3.2 ServerConfig

```go
func ServerConfig(certPEM, keyPEM []byte) (*tls.Config, error)
```

**算法：**
1. `tls.X509KeyPair(certPEM, keyPEM)` 得到 `tls.Certificate`
2. 返回：
```go
&tls.Config{
    Certificates: []tls.Certificate{cert},
    MinVersion:   tls.VersionTLS12,
}
```

### 3.3 ClientConfig

```go
func ClientConfig(caPEM []byte, skipVerify bool) (*tls.Config, error)
```

**算法：**
- `skipVerify=true` → `InsecureSkipVerify: true`
- `caPEM != nil` → 将 caPEM 加入 `x509.CertPool`，设置 `RootCAs`
- `MinVersion: tls.VersionTLS12`

## 4. server/lb.go 详细设计

### 4.1 Node 结构体

```go
type Node struct {
    id          string           // RemoteAddr 字符串，唯一标识
    conn        net.Conn         // 底层 TLS 连接
    writeMu     sync.Mutex       // 写帧互斥，防止帧交叉
    activeConns int32            // 原子操作，当前活跃业务连接数
    pending     map[uint32]chan<- *proto.Frame  // reqID → 响应投递通道
    pendingMu   sync.RWMutex     // 保护 pending map
    done        chan struct{}     // 节点退出信号
}
```

**关键方法：**

```go
// Send 线程安全地发送帧
func (n *Node) Send(f *proto.Frame) error {
    n.writeMu.Lock()
    defer n.writeMu.Unlock()
    return proto.WriteFrame(n.conn, f)
}

// RegisterPending 注册一个 req_id 的响应通道
func (n *Node) RegisterPending(reqID uint32, ch chan<- *proto.Frame) {
    n.pendingMu.Lock()
    n.pending[reqID] = ch
    n.pendingMu.Unlock()
}

// DeregisterPending 注销并返回通道
func (n *Node) DeregisterPending(reqID uint32) (chan<- *proto.Frame, bool) {
    n.pendingMu.Lock()
    ch, ok := n.pending[reqID]
    delete(n.pending, reqID)
    n.pendingMu.Unlock()
    return ch, ok
}

// Deliver 投递一帧到对应的 pending channel（非阻塞）
func (n *Node) Deliver(f *proto.Frame) {
    n.pendingMu.RLock()
    ch, ok := n.pending[f.ReqID]
    n.pendingMu.RUnlock()
    if ok {
        select {
        case ch <- f:
        default: // channel 满则丢弃（超时场景）
        }
    }
}

// CloseAll 断线时关闭所有 pending channel
func (n *Node) CloseAll() {
    n.pendingMu.Lock()
    for id, ch := range n.pending {
        close(ch)  // 触发 tunnel→nginx goroutine 退出
        delete(n.pending, id)
    }
    n.pendingMu.Unlock()
}
```

### 4.2 Pool 结构体

```go
type Pool struct {
    mu    sync.RWMutex
    nodes []*Node
}

// Add 注册节点
func (p *Pool) Add(n *Node) {
    p.mu.Lock()
    p.nodes = append(p.nodes, n)
    p.mu.Unlock()
}

// Remove 注销节点
func (p *Pool) Remove(id string) {
    p.mu.Lock()
    for i, n := range p.nodes {
        if n.id == id {
            p.nodes = append(p.nodes[:i], p.nodes[i+1:]...)
            break
        }
    }
    p.mu.Unlock()
}

// Pick 最少活跃连接选择
// 算法：O(n) 遍历，选 activeConns 最小节点
// 无节点时返回 ErrNoNode
func (p *Pool) Pick() (*Node, error) {
    p.mu.RLock()
    defer p.mu.RUnlock()
    if len(p.nodes) == 0 {
        return nil, ErrNoNode
    }
    best := p.nodes[0]
    min  := atomic.LoadInt32(&best.activeConns)
    for _, n := range p.nodes[1:] {
        if c := atomic.LoadInt32(&n.activeConns); c < min {
            min, best = c, n
        }
    }
    return best, nil
}
```

## 5. server/tunnel.go 详细设计

**职责：** 监听隧道端口，管理 ts-client 连接生命周期。

### 5.1 StartTunnel 函数

```go
func StartTunnel(addr string, tlsCfg *tls.Config, pool *Pool,
                 reqIDGen *uint32, logger *slog.Logger)
```

**流程：**
```
tls.Listen("tcp", addr, tlsCfg)
loop:
    conn, err = ln.Accept()
    if err → log.Warn, continue
    node = NewNode(conn)
    pool.Add(node)
    log.Info("client connected", node.id)
    go serveNode(node, pool, logger)
```

### 5.2 serveNode 函数

```go
func serveNode(node *Node, pool *Pool, logger *slog.Logger)
```

**流程：**
```
defer:
    pool.Remove(node.id)
    node.conn.Close()
    node.CloseAll()           // 触发所有等待 goroutine 退出
    log.Info("client disconnected", node.id)

设置读超时定时器（90s，每收到心跳重置）
loop:
    frame, err = proto.ReadFrame(node.conn)
    if err → return
    重置超时定时器
    switch frame.Type:
        TypeHeartbeat:
            node.Send(&Frame{Type: TypeHeartbeatAck})
        TypeDataRsp, TypeClose:
            node.Deliver(frame)
        default:
            log.Warn("unknown frame type")
```

## 6. server/proxy.go 详细设计

**职责：** 监听业务端口，接受 nginx 连接，驱动双向透传。

### 6.1 StartProxy 函数

```go
func StartProxy(addr string, pool *Pool, timeout time.Duration,
                reqIDGen *uint32, logger *slog.Logger)
```

**流程：**
```
net.Listen("tcp", addr)
loop:
    conn, err = ln.Accept()
    if err → log.Warn, continue
    go handleConn(conn, pool, timeout, reqIDGen, logger)
```

### 6.2 handleConn 函数

```go
func handleConn(nginxConn net.Conn, pool *Pool,
                timeout time.Duration, reqIDGen *uint32,
                logger *slog.Logger)
```

**流程：**
```
defer nginxConn.Close()

node, err = pool.Pick()
if err:
    log.Warn("no available client")
    return

reqID = atomic.AddUint32(reqIDGen, 1)
atomic.AddInt32(&node.activeConns, +1)

// 响应帧通道，带缓冲防止阻塞
respCh = make(chan *proto.Frame, 32)
node.RegisterPending(reqID, respCh)

defer:
    node.DeregisterPending(reqID)
    atomic.AddInt32(&node.activeConns, -1)
    // 通知 ts-client 关闭此 req_id
    node.Send(&Frame{ReqID: reqID, Type: TypeClose})

// 超时控制
ctx, cancel = context.WithTimeout(context.Background(), timeout)
defer cancel()

// nginx→tunnel goroutine
go func():
    buf = make([]byte, 32*1024)
    loop:
        nginxConn.SetReadDeadline(time.Now().Add(timeout))
        n, err = nginxConn.Read(buf)
        if n > 0:
            node.Send(&Frame{ReqID:reqID, Type:TypeDataReq, Payload:buf[:n]})
        if err → cancel(); return

// tunnel→nginx goroutine（当前 goroutine）
loop:
    select:
        case <-ctx.Done(): return
        case frame, ok = <-respCh:
            if !ok: return   // node 断线，channel 被 CloseAll 关闭
            if frame.Type == TypeClose: return
            nginxConn.Write(frame.Payload)
```

## 7. client/tunnel.go 详细设计

**职责：** 管理与 ts-server 的 TLS 连接，心跳，帧分发。

### 7.1 Connect 函数（含重连循环）

```go
func Connect(serverAddr string, localAddr string,
             tlsCfg *tls.Config, heartbeat time.Duration,
             logger *slog.Logger)
```

**流程：**
```
backoff = 1s
loop:
    conn, err = tls.Dial("tcp", serverAddr, tlsCfg)
    if err:
        log.Warn("connect failed", err)
        sleep(backoff)
        backoff = min(backoff*2, 60s)
        continue
    backoff = 1s
    log.Info("connected to server")

    // 心跳 goroutine
    stopHeartbeat = make(chan struct{})
    go heartbeatLoop(conn, heartbeat, stopHeartbeat)

    // pending map（req_id → chan *Frame，用于 tunnel→local 数据投递）
    pending = make(map[uint32]chan *proto.Frame)
    pendingMu = sync.Mutex{}

    // 读帧主循环
    err = readLoop(conn, localAddr, pending, &pendingMu, logger)
    close(stopHeartbeat)
    conn.Close()
    // 清理所有 pending channel
    pendingMu.Lock()
    for _, ch := range pending { close(ch) }
    pendingMu.Unlock()

    log.Warn("disconnected, reconnecting...", err)
    sleep(backoff)
    backoff = min(backoff*2, 60s)
```

### 7.2 heartbeatLoop 函数

```go
func heartbeatLoop(conn net.Conn, interval time.Duration,
                   stop <-chan struct{}, writeMu *sync.Mutex)
```

**流程：**
```
ticker = time.NewTicker(interval)
loop:
    select:
        case <-stop: return
        case <-ticker.C:
            writeMu.Lock()
            proto.WriteFrame(conn, &Frame{Type: TypeHeartbeat})
            writeMu.Unlock()
```

### 7.3 readLoop 函数

```go
func readLoop(conn net.Conn, localAddr string,
              pending map[uint32]chan *proto.Frame,
              pendingMu *sync.Mutex, writeMu *sync.Mutex,
              logger *slog.Logger) error
```

**流程：**
```
loop:
    frame, err = proto.ReadFrame(conn)
    if err → return err

    switch frame.Type:
        TypeHeartbeatAck: // 忽略
        TypeDataReq:
            // 新请求，建本地连接
            ch = make(chan *proto.Frame, 32)
            pendingMu.Lock()
            pending[frame.ReqID] = ch
            pendingMu.Unlock()
            go forward.HandleRequest(conn, writeMu, frame.ReqID,
                                     frame.Payload, localAddr, ch, logger)
                → 结束时 pendingMu.Lock(); delete(pending, reqID); pendingMu.Unlock()

        TypeClose:
            pendingMu.Lock()
            ch, ok = pending[frame.ReqID]
            delete(pending, frame.ReqID)
            pendingMu.Unlock()
            if ok: close(ch)

        TypeDataRsp: // 不应收到，忽略
```

## 8. client/forward.go 详细设计

**职责：** 对单个 req_id 进行本地服务的双向透传。

### 8.1 HandleRequest 函数

```go
func HandleRequest(
    tunnelConn net.Conn,
    writeMu    *sync.Mutex,
    reqID      uint32,
    initData   []byte,     // 第一帧 DataReq 的 payload（立即写入本地）
    localAddr  string,
    inCh       <-chan *proto.Frame,  // tunnel→local 数据通道
    logger     *slog.Logger,
)
```

**流程：**
```
// 发送帧的辅助函数（加写锁）
sendFrame = func(f *proto.Frame) error:
    writeMu.Lock()
    err = proto.WriteFrame(tunnelConn, f)
    writeMu.Unlock()
    return err

localConn, err = net.DialTimeout("tcp", localAddr, 10s)
if err:
    log.Error("connect local failed", err)
    sendFrame(&Frame{ReqID: reqID, Type: TypeClose})
    return

defer localConn.Close()

// 写入第一批数据
localConn.Write(initData)

ctx, cancel = context.WithCancel(context.Background())
defer cancel()

// local→tunnel goroutine
go func():
    buf = make([]byte, 32*1024)
    loop:
        n, err = localConn.Read(buf)
        if n > 0:
            sendFrame(&Frame{ReqID:reqID, Type:TypeDataRsp, Payload:buf[:n]})
        if err:
            cancel()
            return

// tunnel→local（当前 goroutine）
loop:
    select:
        case <-ctx.Done():
            sendFrame(&Frame{ReqID: reqID, Type: TypeClose})
            return
        case frame, ok = <-inCh:
            if !ok:   // channel 被关闭（服务端断线）
                return
            if frame.Type == TypeClose:
                return
            localConn.Write(frame.Payload)
```

## 9. server/main.go 和 client/main.go

### 9.1 server/main.go 关键逻辑

```go
func main() {
    tunnelPort := flag.String("tunnel-port", "9000", "...")
    httpPort   := flag.String("http-port",   "9001", "...")
    certFile   := flag.String("cert",        "",     "...")
    keyFile    := flag.String("key",         "",     "...")
    timeout    := flag.Duration("timeout",   60*time.Second, "...")
    logLevel   := flag.String("log-level",   "info", "...")
    flag.Parse()

    // 日志
    logger = slog.New(slog.NewTextHandler(os.Stdout, &slog.HandlerOptions{
        Level: parseLevel(*logLevel),
    }))

    // TLS 证书
    var certPEM, keyPEM []byte
    if *certFile != "" {
        certPEM, _ = os.ReadFile(*certFile)
        keyPEM,  _ = os.ReadFile(*keyFile)
    } else {
        certPEM, keyPEM, _ = tlsutil.GenerateSelfSigned()
        logger.Info("使用自动生成的自签名证书")
    }
    tlsCfg, _ := tlsutil.ServerConfig(certPEM, keyPEM)

    pool      := &lb.Pool{}
    var reqIDGen uint32

    // 启动隧道监听
    go tunnel.StartTunnel(":"+*tunnelPort, tlsCfg, pool, &reqIDGen, logger)
    logger.Info("tunnel listening", "port", *tunnelPort)

    // 启动业务监听（阻塞）
    logger.Info("proxy listening", "port", *httpPort)
    proxy.StartProxy(":"+*httpPort, pool, *timeout, &reqIDGen, logger)
}
```

## 11. 帧时序图

### 11.1 正常请求

```
ts-server                              ts-client
    │                                      │
    │←─────────── HEARTBEAT ───────────────│
    │──────────── HEARTBEAT_ACK ──────────►│
    │                                      │
    │──────────── DATA_REQ(id=1) ─────────►│
    │                                      │──► 本地服务
    │                                      │◄── 响应
    │◄─────────── DATA_RSP(id=1) ──────────│

```
ts-server                              ts-client

## 12. 性能设计

### 12.1 缓冲区大小

| 场景 | 缓冲区 | 说明 |
|------|--------|------|
| nginx/本地服务 read buf | 32KB | 平衡内存与吞吐 |
| respCh / inCh channel | 32 帧 | 防止写阻塞 |
| WriteFrame 内部 buf | 9 + payload | 一次 syscall 写完整帧 |

### 12.2 零拷贝原则

- Read → Frame.Payload → Write 全程使用同一 slice，不额外复制
- `io.ReadFull` 直接读入目标 slice，无中间 buffer
